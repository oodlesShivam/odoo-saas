Real Multi Website (Online Event's Tickets extension)

Configure Event's Tickets per website

Key features:
  • All features from Real Multi Website module
  • Shows Event's Tickets per website

How It Works

Let's say we have 2 websites:

  • example1.com
  • example2.com
Go to Events → Events menu and select one of your event record to configure. At Allowed Websites select the websites, where your event should be available on, e.g. example1.com. Note that you can only select chosen company's websites. To select any websites leave Company field empty.

Open the /events page of your website and see the configured events. Note that without this module there is already a rule based on event's company - you can only see events with no company specified or events whose company is child of current user's company. But with this module installed you have also the ability to filter events per website - you can only see events with no Allowed Websites specified or current website's events
