E-commerce "Buy now"

Accelerate sales with "Buy now" button

Usage

Add a settings to Product, that allows to sale:

  • via usual "Add to cart" button only
  • via "Buy Now" button only
  • both buttons

After clicking on "buy now" at web store:

  • cart is cleared and the product is added
  • user is redirected to /shop/checkout page
  • page /shop/cart is still availabe, but it doesn't have "continue shopping button"

"Buy Now" button only
