Pickup and pay at store

Simplify checkout process by excluding shipping and\or payment information

On checkout step customer selects the shipping and billing variant. If chosen variant doesn't require "payment", the payment step will be skipped. All variants and the corresponding fillable fields are introduced below.

Pickup and pay at store

Pickup at store but pay now

Pay now and get delivery

Pay on delivery

Administrator may configure which of these option need to be used.
