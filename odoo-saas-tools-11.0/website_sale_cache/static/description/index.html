E-commerce Category Cache

Use this module to greatly accelerate the loading of a page with a large number of product categories

Usage

Open your website shop and load the page containing products. After the first load, the website product categories are cached and the page loading speed will be increased.

Additional information

When multiple odoo workers are used, the cache is updated for each one separately. So the page may be loaded slowly before updating the cache for each worker. Once the latter is done, the page will be loading faster, as far as the advantages of caching allow. After update the list of categories (deleting a category, updating or creating a new one), the cache is updated and once again the speed will increase after the first load.

Note

Cache is updated every time on creation, editing or deletion the product public category model instance. It is recommended to temporarily uninstall the module in case of importing a large number of categories.
